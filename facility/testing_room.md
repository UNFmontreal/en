# Testing room

The testing room (R0743) is located within the Institut universitaire de gériatrie de Montréal (IUGM).

## Opening hours

**Regular opening hours**

  Monday to Friday, except public holidays

```
9:00 AM-12:00 PM and 1:00 PM-5:00 PM

if you need technical assistance from UNF staff
 OR
if you do not have an access card to enter the UNF
```

**Outside regular hours**

```
7 days a week, 24 hours a day

if you do not need technical assistance from UNF staff
```

## Note

There is no fee for using the testing room. However, it must be used in connection with ongoing projects at the UNF or in preparation for them.
